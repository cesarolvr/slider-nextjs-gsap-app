### Getting started

**Run:**
```javascript
yarn
yarn dev
```
This will start application in development mode.

**Build and serve:**
```javascript
yarn build
yarn start
```
This will generate static files and serve in a local server.

**How to deploy new version:**
Just run `vercel deploy`.
